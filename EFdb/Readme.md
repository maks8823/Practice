Задание 1.

В ходе задания необходимо создать пустое консольное приложение. Добавить в него библиотеки и / или пакеты необходимые для работы с EntityFramework Core.
В корне проекта создать директорию Models, в которой будут размещены сущности хранимые в базе данных согласно вариантам.
В проекта создать директорию Data, в ней необходимо описать наследника класса DbContext для работы с базой данных на основе ваших моделей.
Реализовать миграции. И код который демонстрирует работу с БД (создает данные, выбирает, обновляет, удаляет). Все запросы снабдить комментариями, которые описывают намерения, что предполагалось.
